CARS Management System

The CARS Management System is a web application that was designed and developed for Calgary Animal Rescue Society (CARS). This app helps manage the core functions of nonprofit animal rescue organizations, including adoption forms management, and events & volunteer tracking. The system can be accessed via any modern desktop or mobile browser.

Designed and developed using Angular, Bootstrap, Spring MVC, and MySQL.

Inventory Tracker

The Inventory Tracker is a software that can help to track personal items in the home, office, or at a self-storage facility. It stores user data on a server and can be accessed from any modern desktop or mobile browser. This app will feature a user-friendly interface, online registration, password protection, and two-factor authentication support.

Designed and developed using HTML5, CSS3, Java, and SQL.
