Inventory Tracker Application

Inventory Tracker is software that anyone can use to track personal items in the home, office, or at a self-storage facility. This web-based tool features user-friendly interface, online registration, password protection, and two-factor authentication support.

The application is designed using HTML5, CSS3, Java, and MySQL.
