CARS Management System

CARS Management System was designed specifically for Calgary Animal Rescue Society (CARS). Developed to minimize the labour work associated with handling hundreds of applications for adoption of rescued animals, becoming a foster or on-site volunteering, and signing up for CARS’ seasonal events.

Designed and developed using Angular, Bootstrap, Spring MVC, and MySQL.

Client

The ultimate goal of this project is to facilitate the organization’s resource management and maximize its capacity to rescue as many animals as possible. The app to aid general and specific administrative tasks, this app also enables CARS staff to easily navigate through the database of rescued animals, current and potential volunteer workers & foster families, and the facility’s vacancy, as well as its capacity. Moreover, we are also working on the CARS’ website as a part of this project. Once the system is operational, anyone will be able to see the rescued animals on the website and complete the application for adoption on-line. Both the app and website can be accessed from a desktop computer or a mobile device.

E-Vault Inventory Tracker

The E-Vault Inventory Tracker app is a software that anyone can use to help track personal items in the home, office, or at a self-storage facility. It stores user data on a server and can be accessed from any device. This app features user-friendly interface, online registration, password protection, and two-factor authentication support.

Designed and developed using Angular, TypeScript, NgRx, RxJS, .NET, C#, SQL, HTML, CSS, Bootstrap and Git.

The purpose of the system is to make it easier and more efficient than ever before to access stored items. The application is accessible from desktop and mobile devices.
